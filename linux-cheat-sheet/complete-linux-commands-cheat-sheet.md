# Complete Linux Commands Cheat-Sheet

This document gives you the detailed documentation of all the Linux Commands.

## Basic File and Directory Operations

### ls

It is used to list the files in the current directory.

### cd
It is used to navigate the file system (change directory).

### pwd

It is used to know your current location in the file system.

### mkdir
It is used to create directory or nested directories.

### touch
It is used to create new files, modify file timestamps, and manage file operations efficiently.

### cp
It is used to copy files, directories, and preserve file attributes and timestamps effectively.

### mv
It is used to rename files, move multiple files, and perform other file management tasks.

### rm
It is used to remove files and directories, handle confirmation prompts, and force removal for efficient file management.

### ln
It is used to create hard and symbolic links.

### cat
It is used to concatenate and display text files, as well as append content to existing files, enhancing your basic file and directory operations skills.

### less
It is used to navigate through text files, search and highlight content.

### more
It is used to navigate and search through text files, and customize its behavior for efficient file viewing.

### tree
It is used to display the hierarchical directory structure of your current directory and all its subdirectories and files within it.

### du
It is used to measure disk usage.

### df
It is used for monitoring disk usage.

### stat
It is used to retrieve file metadata, and analyze file permissions and ownership.

### find
It is used to search for files by name, file type, and combine find with other commands for advanced file system operations.

### locate
It is used for quickly searching and finding files on your system.

### updatedb
It is used to update the locate database for efficient file searches.

### which
It is used to locate the path of executable files.

### whereis
It is used to locate executable files, source code, and manual pages on your system.

### file
It is used to identify file types, and handle compressed files effectively.

### od
It is used perform hexadecimal dumps of files.

### mktemp
It is used to create secure temporary files.

### basename
It is used to extract filenames from full paths, and combine it with other commands for efficient file and directory operations.

### dirname
It is used to retrieve the directory name from a file path.

### dirs
It is used to manage the directory stack, and to navigate directories efficiently.

### mc
It is used to perform essential file and directory operations using this powerful command-line tool.

### readlink
It is used resolving symbolic links.

### rename
It is used to rename files and directories.

### rmdir
It is used to create and delete empty directories, and remove non-empty directories.

### shred
It securely deletes files by overwriting their contents multiple times, ensuring complete data erasure.

### chattr
It is used to modify file attributes, and protect important files with the immutable attribute.

### lsattr
It allows you to view and manage file attributes.

### cksum
It is used to calculate checksums for files, and verify file integrity.

### cmp
It is used to compare text and binary files.

### mtools
It is used for managing floppy disk images on Ubuntu 22.04.

### mcopy
It is used to copy files and directories.

### mdel
It is used for creating and managing multiple directories.

### mdir
It is used to create and manage directories.

### mmove
It is used to effectively move files and directories.

### mread
It is used for efficient file reading.

### mren
It is a powerful tool for renaming multiple files efficiently.

### mshowfat
It is used to analyze and troubleshoot FAT-based storage devices.

### mtype
It displays the contents of an MS-DOS file on a Unix/Linux system

### mattrib
It is used for modifying file attributes, managing multiple files and directories.

### mmd
It is used to create Markdown files, and convert them to HTML and PDF formats for various use cases.

### mrd
It is used for managing directories.

### mzip
It is used to compress and extract files and directories using this versatile tool.

### mtoolstest
It is used to verify the mtools configuration files and display the cumulative configuration settings

### tee
It is used to redirect output to both a file and the terminal, and append output to existing files.

### read
It is used to capture user input, store it in variables, and validate the input.
